# Dashboard Boundary

Future dashboard code belongs here.

The UI should read normalized records from `data/history.json` or from an API that serves the same schema. It should not reconstruct vault value, supply, burn events, prices, NAV, backing, or premium directly in React/browser code.

Planned KPI modules:

- Vault Value
- NAV
- Treasury Backing
- Premium vs NAV
- Current Supply

Planned chart modules:

- NAV History
- Treasury Backing History
- Premium History
- Supply History
- Burn History
